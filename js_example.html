1 Sum function

Overview: Returns the sum of two numbers

Parameters:

  • a: Numeric variable
  • b: Numeric variable

1.1 Example

result=sum(5, 10)

function sum(a, b) {
    return a + b;
}

2 Product function

Overview: Returns the product of two numbers

Parameters:

  • a: Numeric variable
  • b: Numeric variable

2.1 Example

result=product(5, 10)

function product(a, b) {
    return a * b;
}
